# 🚀 Projeto Final: It Bank

Este projeto representa a culminação do módulo de Front-end no **Bootcamp Impacta-FullStack**. A aplicação foi desenvolvida para consolidar conceitos avançados de desenvolvimento web moderno, integrando performance, tipagem robusta e design responsivo.

## 🎯 Objetivos do Projeto
Consolidar o aprendizado prático focado em:
* **Componentização** com React.
* **Tipagem** e segurança de código com TypeScript.
* **Roteamento e Performance** utilizando o **App Router** do Next.js.
* **Estilização responsiva** e utilitária com Tailwind CSS.

---

## 🛠️ Tecnologias e Ferramentas
* **Framework:** Next.js.
* **Linguagem:** TypeScript.
* **Estilização:** Tailwind CSS.
---

## 📌 Funcionalidades Implementadas

### 🔐 Acesso e Conta
- [ ] **Setup Inicial:** Configuração do ambiente Next.js com TypeScript.
- [ ] **Sign-in + Sign-up:** Telas de autenticação (estáticas ou integradas).
- [ ] **Criação de Conta:** Fluxo de cadastro para novos usuários.

### 💰 Operações Bancárias
- [ ] **Extrato de Conta:** Visualização histórica de movimentações financeiras.
- [ ] **Depósito:** Interface para entrada de valores na conta.
- [ ] **Saque:** Interface com validação para retirada de valores.
- [ ] **Transferência:** Fluxo para envio de valores entre contas.

## 🏃 Como Executar Localmente

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/nome-do-projeto.git
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Acesse no navegador:**
   `http://localhost:3000`
